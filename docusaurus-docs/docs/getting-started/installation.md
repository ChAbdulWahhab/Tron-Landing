# Installation

## Windows (Standard Setup)

TRON QA Suite is a desktop application. You don't need to run any terminal commands to install it.

1. **Download the Installer**: Go to [tronq.vercel.app](https://tronq.vercel.app/) and click **Get TRON for Windows**.
2. **Run the Installer**: Locate the downloaded file (e.g., `TRONSetup-v3.exe`) and double-click it.
3. **SmartScreen Warning**: Since TRON is an independent tool, Windows might show a "SmartScreen" warning. Click **More Info** and then **Run Anyway**.
4. **Completion**: Once the setup finishes, TRON will launch automatically, and you can find it in your Start Menu.

## First Run: Python Dependencies

TRON uses a powerful Python engine to perform deep scans. On your first run, the app will check if the required engine is ready.

- If you see a prompt asking to install dependencies, click **Install Now**.
- Alternatively, you can go to **Tools → Install Python Dependencies** within the app at any time.

## Requirements

- **OS**: Windows 10 or 11.
- **Browser**: Google Chrome should be installed on your system for advanced JavaScript and mobile rendering tests.
- **Internet**: Required for the initial setup and for scanning live websites.

## If installation fails

- **Antivirus**: Some antivirus programs might block the installer. Temporarily disable them or add an exception for TRON.
- **Permissions**: Ensure you have administrator rights if the installer asks for them.
- **Support**: See [Troubleshooting](../operations/troubleshooting.md) for more help.
